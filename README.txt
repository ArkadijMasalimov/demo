# Название проекта
ООО «Канцелярия» - магазин по продаже товаров для автомобилистов
## Запуск приложения
Для запуска приложения переходим в каталог ООО_Канцелярия1\bin\Debug и запускаем файл ООО_Канцелярия1.exe 
## Средства разработки
- MySql WorkBench
- Visual Studio 2019
- C#
